# Foot-Shop E-commerce Website

## Description
Foot-Shop is a static e-commerce front-end website built using HTML5 and CSS3.
It showcases a modern sneaker store layout including product listings, promotional
sections, and a shopping cart interface.

This project focuses on front-end structure, layout, and usability.

## Features
- Home page with promotional banners
- Product listings with pricing
- Navigation menu
- Newsletter subscription section
- Responsive layout design

## Technologies Used
- HTML5 (semantic markup)
- CSS3
- Font Awesome (icons)
